# Magnification Overlay (Accessibility)

## How to Use:
1. Download the executable (Titled : **MGT_overlay.exe**)
2. Run the executable. (Check **notes** if virus pops up)
3. Resize the window
4. Select "Start Overlay"
5. Enjoy your eye strain free gaming!

## Notes:
- Microsoft's antivirus software sometimes identifies it as a virus. Just allow the file and don't terminate it.
    1. Select "More Info."
    2. A button will pop up saying "Run Anyway" <- select that.
- If you encounter any visual glitches _(Known Issue)_, please resize the DearImGui window, and the issue will be resolved.
